this is a blendos track for hyprland with dms

after first boot make sure to run ```/init.sh```
that will initialize/reinitialize your hyprland config

The ```blendos-base.yaml``` file is a copy of the one from the blendos repos with the
kernel removed. If you use either
```blendos-base.yaml``` or ```hyprland-custom-kernel.yaml```
make sure you have installed a kernel in your ```system.yaml```
